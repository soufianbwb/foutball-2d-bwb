# 2D Football Game

A simple 2D football game built with Python and Pygame.

## Features

- Two players controlling football players
- Ball physics with collision detection
- Simple AI opponent
- Score tracking
- 60 FPS gameplay

## Installation

```bash
pip install -r requirements.txt
```

## How to Play

Run the game:
```bash
python main.py
```

### Controls

| Control           | Action                       |
|-------------------|------------------------------|
| **W/S**           | Move Player 1 up/down        |
| **UP/DOWN Arrows**| Move Player 2 up/down        |

## Game Rules

- Control your player to kick the ball
- Try to move the ball towards the opponent's goal
- The ball bounces off walls and players
- Simple collision detection with automatic bounce

## Requirements

- Python 3.7+
- Pygame 2.5.2
